# CMUxRIT-TradingComp

Starter repository for CMU x RIT trading competition development.

## Structure

- `src/` — strategy and execution code
- `.gitignore` — common ignores for Python/Go/editor artifacts

## Quick start

```powershell
# from this folder
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

```powershell
# optional if using Go modules
go mod init cmuxrit-tradingcomp
```
